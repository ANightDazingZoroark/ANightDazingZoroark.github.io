Download Prehistoric Rift

You can download the latest version of Prehistoric Rift directly here.

Before playing, please note the following:

  • This addon may not work correctly with other addons that edit player and vanilla mob behaviors
  • You must be on the latest version of Minecraft Bedrock Edition
  • You must turn on Vanilla Experiments, Holiday Creator Features, Custom biomes, Upcoming Creator Features, Gametest Framework, Molang Features, and cheats in your world




Older versions can be downloaded here.

Be sure to check out the wiki every now and then for information. You can even contribute to it if you want to!

0.6.9 Patch Notes

wow funny number patch version

Changes to existing creatures

  • General
    • Attempted to change the models and textures of every single creature in the addon
    • Tier 2 creatures that are in the process of being tamed will now come towards you when you're holding a treat and when they want to eat
    • If you hit a Tier 2 creature that is in the process of being tamed, all progress that you may have made while taming it will be gone and you need to make it trust you again
    • Babies that hatch from any egg you place or obtain from breeding will now be automatically tamed to you
    • Started using the GameTest API to manage quite a lot of events
    • If you place food items in a creature's inventory they will eat them whenever they lose health
    • Temporarily removed costume functionality. All creatures that have been given costumes will revert to their original forms if you update to this version
  • Apatosaurus
    • Welp guess what. The Apatosaurus's platform now has a mini crafting table and a mini furnace. The mini crafting table can be used to craft wood related stuff, plant related stuff, and food related stuff
    • Will now eat azalea and mangrove leaves
    • You no longer need to load up the inventory of an Apatosaurus with ammunition. They will instead use ammunition placed in their inventories instead
  • Dimetrodon
    • Frost and Flame bite attacks no longer have a splash effect
    • Frost and Flame bite attacks should now work on every single mob a Dimetrodon attacks
  • Direwolf
    • Replaced the boulder throw of the werewolf form with a breath attack because The Wolf Among Us is epic
    • Changed how sniffing ability works, an indicator will show up in where a detected entity was at
  • Tyrannosaurus
    • Changed roar sound

Other changes

  • Leafy treats can now be crafted with azalea and mangrove leaves
  • Added different textures for the taxidermy stand based on what type of planks you used to create it
  • Taxidermy stands now require planks + cobblestone to be created
  • Mob events will now only happen when playing hard mode
  • Added new textures created by Russian Soldier

New Items

  • Added a new Journal item, which you can open to read all entries
  • Added wooden spiked wall
  • Added metal spiked wall
  • Added Large Gate
    • A 4 block wide 4 block tall gate
  • Added Behemoth Gate
    • An 8 block wide 8 block tall gate
  • Added Mangrove variants for the ff:
    • Thatch
    • Thatch Block
    • Wooden Table
    • Wooden Chair
    • Wooden Bench

Other new things

  • You can now duplicate blueprints by placing a blueprint and a paper in any crafting menu
  • Added The Flood, a new mob event which spawns in Baryonyxes and Sarcosuchuses and summons a thunderstorm
  • There's a secret new mob that will only spawn by mining ores

Bug Fixes

  • Fixed bug where placing item in Apatosaurus inventory will remove the platform (#103)
  • Fixed creatures being unable to harvest azalea leaves (#152)
