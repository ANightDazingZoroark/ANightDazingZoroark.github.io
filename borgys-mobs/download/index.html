Download Borgy's Mobs

You can download the latest version of Borgy's Mobs directly here.

Before playing, please note the following:

  • This addon may not work correctly with other addons that edit player and vanilla mob behaviors
  • You must be on the latest version of Minecraft Bedrock Edition
  • You must turn on Holiday Creator Features, Advanced Modding Capabilities, Commands, and Upcoming Creator Features in your world




Older versions can be downloaded here.

Be sure to check out the wiki every now and then for information. Sure it's quite empty, but it will get filled up as time goes on, and you can even contribute to it if you want to!

1.2.3 Patch Notes

A new update with some new items

New Items

  • Added Bile Gland
    • Dropped by Bullquids, is somewhat common
    • Can be used to make the Bile Launcher
  • Added Bile Launcher
    • Launches bile that is similar to the bullsquid's projectiles
    • Has 60 durability and a 1 second cooldown, you can repair it with bile glands
  • Added Headcrab Launcher
    • Launches headcrabs
    • Shoot it at mobs to turn them into headcrab zombies
    • Has 30 durability, you can repair it with ender pearls
  • Added Bakunawa Scales
    • A somewhat common drop of the Bakunawa
    • Can be crafted into armor
  • Added Bakunawa Armor
    • Has durability and protection equal to that of diamond armor
    • When worn in its entirety, you will get conduit power, strength, and slow falling

Changes to existing mobs

  • General
    • The indicator that shows up above tamed mobs are more opaque
  • Fast Headcrab (and fast headcrab zombie)
    • Will now run from Sigbins
  • Headcrab (and headcrab zombies)
    • Will now run from Sigbins
  • Houndeye
    • Wild Houndeyes will now go after pigs
  • Poison Headcrab (and poison headcrab zombie)
    • Will now run from Sigbins
  • Sigbin
    • Will now drop gold nuggets upon death
