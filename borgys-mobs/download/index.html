Download Borgy's Mobs

You can download the latest version of Borgy's Mobs directly here.

Before playing, please note the following:

  • This addon may not work correctly with other addons that edit player and vanilla mob behaviors
  • You must be on the latest version of Minecraft Bedrock Edition
  • You must turn on Holiday Creator Features, Advanced Modding Capabilities, Commands, Upcoming Creator Features, and Creation of Custom Biomes in your world




Older versions can be downloaded here.

Be sure to check out the wiki every now and then for information. Sure it's quite empty, but it will get filled up as time goes on, and you can even contribute to it if you want to!
