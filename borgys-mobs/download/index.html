Download Borgy's Mobs

You can download the latest version of Borgy's Mobs directly here.

Before playing, please note the following:

  • This addon may not work correctly with other addons that edit player and vanilla mob behaviors
  • You must be on the latest version of Minecraft Bedrock Edition
  • You must turn on Holiday Creator Features, Advanced Modding Capabilities, Commands, Upcoming Creator Features, and Creation of Custom Biomes in your world




Older versions can be downloaded here.

Be sure to check out the wiki every now and then for information. Sure it's quite empty, but it will get filled up as time goes on, and you can even contribute to it if you want to!

1.3.3 Patch Notes

This update adds compatibility with Prehistoric Rift (specifically version 0.6.1, yay) alongside other small changes. Make sure to put the behavior and resource packs of Prehistoric Rift above the behavior and resource packs of Borgy's Mobs. It should look something like this:

Bonus Features if you use Prehistoric Rift

  • Trikes will gain their rival buff in the presence of wild Bakunawas
  • The perks from using armor from Borgy's Mobs apply properly
  • Most vanilla mobs will fight back properly when forced attacked by a creature from either Prehistoric Rift or Borgy's Mobs
  • The following creatures will now run from a Parasaurolophus's scare ability
    • Houndeye
    • Hoteye
    • All Headcrabs
    • All Headcrab Zombies
    • Sigbin
    • Sarimanok

Other small changes

  • Did minor adjustments to Bakunawa and Icthyosaur navigation
  • Witches now attack headcrabs and headcrab zombies
